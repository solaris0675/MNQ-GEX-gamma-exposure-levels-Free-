# MNQ-GEX-gamma-exposure-levels-Free-
Free NQ/MNQ Gamma Exposure (GEX) Dashboard using live QQQ options data. Calculates Gamma Flip, Call Wall, Put Wall, and key GEX levels across multiple expirations with professional visualizations and CSV exports.
NQ/MNQ GEX Dashboard — A free Python dashboard that calculates and visualizes Nasdaq Gamma Exposure (GEX) levels using live QQQ options data from Yahoo Finance. Features Gamma Flip, Call Wall, Put Wall, top GEX levels, multi-timeframe analysis (0DTE, 1DTE, 7DTE, 14DTE, 30DTE, ALL expirations), interactive charts, and NQ/MNQ price-level conversion. Designed for traders seeking institutional-style gamma analysis without paid options data feeds. Exports CSV reports and generates both histogram-based and price-overlay GEX visualizations.
